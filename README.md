# school-database-project
School database system using MySQL (students, teachers, subjects)

This project is a simple database system for a high school.

## Features
- Manage students data
- Manage teachers data
- Manage subjects
- Perform SQL operations (SELECT, INSERT, UPDATE)

## Technologies
- MySQL

## Data
- 30 Students
- 10 Teachers
- 6 Subjects
